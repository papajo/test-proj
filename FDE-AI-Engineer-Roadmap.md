# Becoming a Forward Deployed Engineer (FDE) for AI: A 90-Day Roadmap

The role of a **Forward Deployed Engineer (FDE) for AI** is rapidly becoming one of the most sought-after positions in the tech industry, offering lucrative salaries ranging from **$200K–$600K+.** It's a hybrid role that combines software development, machine learning expertise, and client-facing skills. As an FDE, you'll bridge the gap between cutting-edge **AI products** and real-world enterprise challenges involving messy data, diverse use cases, and security constraints.

This 90-day accelerated roadmap guides you through the essential skills required to master system deployment, applied AI fluency, and customer-facing engineering.

---

### **Phase 1: Core AI & Enterprise Foundations (Days 1–30)**

To begin your journey, you need to establish foundational knowledge in interacting with AI applications and understanding the reality of deploying systems within enterprise environments.

#### **1. AI Application Basics**
- **Goal:** Build familiarity with modern **Large Language Model (LLM) APIs** like OpenAI, Anthropic, and others.
    - Learn how to leverage AI for data-grounded responses rather than surface-level queries.
    - Master the use of **prompts**, function calls for APIs, and techniques to structure inputs effectively.
    - Explore practical AI tools such as LangChain, Hugging Face, and OpenAI's function calling API.

##### Resources:
- OpenAI documentation: [OpenAI API Overview](https://platform.openai.com/docs/)
- Hugging Face tutorials: [Hugging Face Course](https://huggingface.co/course/chapter1)

#### **2. Enterprise Cloud & Security**
- **Goal:** Deepen your understanding of cloud infrastructure and security, critical for AI deployment in enterprise environments.
    - Familiarize yourself with **cloud platforms** such as AWS, Google Cloud Platform (GCP), and Microsoft Azure.
    - Key topics to master include **VPC peering**, **cross-account IAM roles**, as well as secure data handling and key/secret management.
    - Understand how AI systems interface with client firewalls or secure environments.

##### Resources:
- AWS Cloud Basics: [AWS Documentation](https://aws.amazon.com/documentation/)
- GCP IAM Roles: [Google Cloud IAM Roles](https://cloud.google.com/iam/docs/understanding-roles)
- Azure Security Center Overview: [Azure Security Center](https://learn.microsoft.com/en-us/azure/security-center/)

#### **3. System Design**
- **Goal:** Understand system architecture for quick, secure integrations of AI products with cloud systems, data pipelines, and real-world applications.
    - Learn backend programming languages such as **Python** or **Go**.
    - Study common backend design patterns and practices.
    
##### Resources:
- Python Tutorials: [Real Python](https://realpython.com/)
- Go Programming Language documentation: [Go.dev](https://go.dev/doc/)

---

### **Phase 2: Applied AI Fluency & Agent Workflows (Days 31–60)**

Once you've grasped core AI and enterprise foundations, proceed to leverage applied AI concepts and develop agent workflows. Focus on production-grade implementations.

#### **1. Advanced Retrieval-Augmented Generation (RAG)**
- **Goal:** Move beyond basic tutorials to advanced **production-grade RAG systems**.
    - Understand and implement **vector databases** (e.g., Pinecone, Milvus, Weaviate, or ChromaDB).
    - Learn **chunking strategies**, embeddings (e.g., Sentence Transformers, OpenAI embeddings), and evaluation techniques such as:
        - Hit Rate
        - **Mean Reciprocal Rank** (MRR)
        - Recall@K metrics

##### Resources:
- RAG Overview: [Introduction to Retrieval-Augmented Generation](https://www.pinecone.io/learn/retrieval-augmented-generation/)
- Getting Started with Vector Databases (Pinecone): [Pinecone Documentation](https://www.pinecone.io/docs/)
- Sentence Embedding Models: [Sentence Transformers](https://www.sbert.net/)

#### **2. Agentic Workflows**
- **Goal:** Design workflows that coordinate **multi-agent orchestration**, function routing, and multi-step reasoning.
    - Create modular systems where AI agents interact and resolve complex tasks collaboratively.
    - Focus on real-world examples where diversity in tasks and reasoning is required.

##### Resources:
- LangChain Agents Documentation: [LangChain Agents](https://docs.langchain.com/docs/modules/agents)
- Multi-agent Systems Overview: [Multi-Agent Systems Research](https://www.journals.elsevier.com/journal-of-artificial-intelligence)

---

### **Phase 3: Production & Stakeholder Management (Days 61–90)**

In the final phase, shift your focus onto operational excellence and customer-facing skill sets to ensure that AI systems deliver results in real-world production.

#### **1. Monitoring & Evaluation**
- **Goal:** Build robust monitoring and evaluation tools for deployed AI systems.
    - Implement performance monitoring to track key metrics like latency, operational costs, and hallucination rates.
    - Introduce **guardrails** to mitigate risks such as output errors, unexpected behavior, and memory/resource issues.

##### Resources:
- AI Performance Metrics Overview: [AI Performance Metrics Article](https://www.analyticsvidhya.com/blog/2022/11/how-to-build-reliable-machine-learning-models-using-performance-metrics/)
- Using Guardrails with AI: [Introducing Guardrails for LLMs](https://towardsdatascience.com/introducing-guardrails-for-large-language-models-with-python-d9a514d618fa)

#### **2. Consulting & Scoping**
- **Goal:** Develop expertise in stakeholder management and building sustainable, production-ready AI systems.
    - Learn how to uncover real-world enterprise problems behind vague client requests.
    - Deliver continuous and actionable solutions rather than unfinished prototypes.

##### Resources:
- Effective Stakeholder Management: [Stakeholder Management Guide](https://www.projectmanager.com/blog/stakeholder-management-step-guide)
- Consulting Skills for Engineers: [Consulting Skills Blog](https://www.consultingsuccess.com/)

---

## Key Takeaways

1. FDEs for AI play a crucial role in developing production-ready AI applications while solving enterprise-level problems and managing client requirements.
2. The 90-day roadmap emphasizes progressing through **three phases**:
   - **Core AI & Enterprise Foundations** (Days 1–30): Build foundational skills in AI applications, enterprise cloud/security, and system design.
   - **Applied AI Fluency & Agent Workflows** (Days 31–60): Master advanced techniques like retrieval-augmented generation and multi-agent systems for production-ready AI workflows.
   - **Production & Stakeholder Management** (Days 61–90): Learn monitoring tools, implement guardrails, and hone consulting/scoping skills to handle client-facing challenges and deliver actionable solutions.
3. FDEs should possess a blend of technical, applied AI, and consulting skills to excel in this role.
