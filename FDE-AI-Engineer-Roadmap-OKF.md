# FDE AI Engineer Roadmap

## Overview

90-day accelerated roadmap to become a Forward Deployed Engineer (FDE) for AI. Covers system deployment, applied AI fluency, and customer-facing engineering skills. Salary range: $200K–$600K+.

**Type**: Learning Roadmap
**Duration**: 90 days (3 phases)
**Target Role**: Forward Deployed Engineer (AI)

## Objectives

1. Master AI application basics (LLM APIs, prompts, function calls)
2. Understand enterprise cloud & security (AWS, GCP, Azure)
3. Build production-grade RAG systems with vector databases
4. Design multi-agent orchestration workflows
5. Implement monitoring, guardrails, and stakeholder management

## Key Results

- [ ] Complete Phase 1 (Days 1-30): Core AI & Enterprise Foundations
- [ ] Complete Phase 2 (Days 31-60): Applied AI Fluency & Agent Workflows
- [ ] Complete Phase 3 (Days 61-90): Production & Stakeholder Management
- [ ] Build 3 portfolio projects (one per phase)
- [ ] Pass FDE interview case study (C.A.S.E. framework)

## Phase Breakdown

### Phase 1: Core AI & Enterprise Foundations (Days 1–30)
- AI Application Basics (LLM APIs, prompts, function calls)
- Enterprise Cloud & Security (VPC, IAM, secrets)
- System Design (Python/Go, backend patterns)

### Phase 2: Applied AI Fluency & Agent Workflows (Days 31–60)
- Advanced RAG (vector DBs, chunking, embeddings)
- Agentic Workflows (multi-agent orchestration)

### Phase 3: Production & Stakeholder Management (Days 61–90)
- Monitoring & Evaluation (latency, costs, guardrails)
- Consulting & Scoping (stakeholder management)

## Interview Framework: C.A.S.E.

1. **Clarify** - Data volume, security, definition of done
2. **Architect** - Data flow using cloud primitives
3. **Solve** - Identify "Delta" (what product doesn't do)
4. **Evaluate** - Prove AI isn't hallucinating, monitor performance
